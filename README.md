# Plotting

#### Для работы требуется Microsoft.NET Framework

Примеры построения графиков функций на языке программирования PascalABC.NET с 
использованием модуля GraphWPF

`g1.pas` -- наивный метод без преобразования системы координат

`g2.pas` -- построение параболы с нанесением масштабной сетки

`gsin.pas` -- построение синусоиды и косинусоиды в одних осях с нанесением масштабной сетки
